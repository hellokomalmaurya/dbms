SQL - Date Functions

The Date & time functions are built-in functions in the SQL server that allows you to manipulate and perform operations on date and time values in the SQL server.

  • DAY()-This function is used to get the day of the month for a specified date.

  • DATENAME()-This function is used to retrieve a specified part of a date or time value as string.

  • YEAR()-This function is used to get an integer number that represents the year of the given date.

  • CURRENT_TIMEZONE()-This function is used to retrieve the current time zone offset from Coordinated Universal Time (UTC).

  • DATEADD()-Is used to add a specific number of intervals to a given date or time value.

  • DATETRUNC()-Is used to truncate a date or time value to a specified datepart (such as year, month, day, etc.).

  • SYSDATETIME()-This function is used to get the date and time of the SQL Server.

SQL - String Functions

SQL string functions are used primarily for string manipulation. The following table details the important string functions:

  • CHAR()-Returns the character for each integer passed

  • ASCII()-Returns numeric value of left-most character

  • CONCAT()Returns concatenated string

  • LOWER()-Returns returns the lower case of the given string.

  • DREVERSE()-Returns a reversed string.

  • UPPER()-returns a string that has all the characters in upper case.

  • REPLACE()-Returns a new string by replacing all the occurrences of the particular part of a string (substring) with a specified string.

SQL - Numeric Functions

SQL numeric functions are used primarily for numeric manipulation and/or mathematical calculations. The following table details the numeric functions :

  • FLOOR()-Returns the largest integer value that is not greater than passed numeric expression.

  • ABS()-Returns the absolute value of numeric expression.

  • POWER()-Returns the value of one expression raised to the power of another expression

  • ROUND()-Returns numeric expression rounded to an integer. Can be used to round an expression to a number of decimal points

  • SQRT()-Returns the non-negative square root of numeric expression.

  • SIN()-Returns the sine of numeric expression given in radians.

  • TAN()-Returns the tangent of numeric expression expressed in radians.
