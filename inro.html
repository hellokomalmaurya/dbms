PL/SQL - Overview

The PL/SQL programming language was developed by Oracle Corporation in the late 1980s as
procedural extension language for SQL and the Oracle relational database. Following are certain notable facts about PL/SQL

  • PL/SQL is a completely portable, high-performance transaction-processing language.

  • PL/SQL provides a built-in, interpreted and OS independent programming environment.

  • PL/SQL can also directly be called from the command-line SQL*Plus interface.

  • Direct call can also be made from external programming language calls to database.

Features of PL/SQL

PL/SQL has the following features:
  • PL/SQL is tightly integrated with SQL.

  • It offers numerous data types.

  • It offers extensive error checking.

  • It offers a variety of programming structures.

  • It supports structured programming through functions and procedures.

  • It supports object-oriented programming.

  • It supports the development of web applications and server pages.
