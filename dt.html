PL/SQL - Data Types

In this chapter, we will discuss the Data Types in PL/SQL. The PL/SQL variables, constants and parameters must have a valid data type,
which specifies a storage format, constraints, and a valid range of values.
We will focus on the SCALAR and the LOB data types in this chapter. The other two data types will be covered in other chapters.
