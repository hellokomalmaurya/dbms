The SQL INSERT INTO Statement

The SQL INSERT INTO Statement is used to add new rows of data into a table in the database. Almost all the RDBMS provide this SQL query to add the records in database tables.
Each value in the records we are inserting in a table using this statement should be of the same datatype as the respective column and satisfy the constraints of the column (if any).
The values passed using an insert statement should match the number of columns in the table or, the number of columns mentioned in the current query.
If any of these conditions are not satisfied, this statement generates an error.

Syntax

syntax of the SQL INSERT INTO statement which are shown below:
INSERT INTO TABLE_NAME 
                VALUES (value1,value2...valueN);

Example

To see an example, let us create a table with name CUSTOMERS in the MySQL database using the CREATE TABLE statement as shown below:
INSERT INTO CUSTOMERS 
                VALUES (7, 'Muffy', 24, 'Indore', 10000.00 );

Verification

To check if the records are inserted into the CUSTOMERS table, use the SELECT query
SELECT * FROM CUSTOMERS;
The table will be displayed with all the records included in it.
